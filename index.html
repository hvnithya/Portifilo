    Cervical Cancer Prediction and Detection using Machine Learning


    This project utilises image classification to identify cervical cancer using MATLAB. The technique separates malignant, precancerous, and normal cells from preprocessed cervical pictures by using a Convolutional Neural Network (CNN) to extract important features. The trained model exhibits great sensitivity and accuracy, making it a promising tool for automated early detection of cervical cancer.
    Result: The code performed better by 98%

    Twitter Hate Speech Detection


    Twitter hate speech detection analyses tweets using machine learning algorithms to identify content that promotes hatred, prejudice, or damage. These systems use natural language processing and pattern recognition to discriminate between objectionable and legitimate statements of opinion. The technique entails building models that have been trained on labelled datasets, allowing the computer to categorise tweets as hate speech or non-hate speech. Continuous refinement and ethical concerns are essential for improving accuracy and reducing biases in spotting harmful information on the platform.

    Book Recommendation system


    Using the KNN algorithm and user-book interaction data, I created a personalised book recommendation system. This project included data preprocessing, creating a user-book matrix, computing cosine similarity between users, and selecting k-nearest neighbours. The technology successfully recommends books to consumers based on the tastes of like readers, hence improving the overall reading experience.
